# Installation

This guide will help you install Mk Notes on your local machine.

---

## Requirements
- Node.js version 20.0 or higher.

## Installation

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="npm" default>
    ```bash
      npm install -g mk-notes
    ```
  </TabItem>
  <TabItem value="yarn">
    ```bash
      yarn global add mk-notes
    ```
  </TabItem>
</Tabs>

To verify that Mk Notes is installed correctly, run the following command:

```bash
  mk-notes --version
```

### Issues with installation ?

Ask for help on our [GitHub repository](https://github.com/Myastr0/mk-notes/issues), or on our Discord server

If the installation was successful, you should see the version of Mk Notes that you have installed.
